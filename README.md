# 📚 DERS YÖNETİM VE TAKİP SİSTEMİ

Bu proje, C++ kullanılarak geliştirilmiş basit bir **öğrenci, ders ve derslik yönetim sistemidir**.  
Sistem, dosyalardan veri okuyarak işlemler yapar ve sonuçları raporlar.

---

## 🚀 Özellikler

- 📂 Dosya tabanlı veri okuma
- 👨‍🎓 Öğrenci bilgilerini işleme
- 📖 Ders bilgilerini analiz etme
- 🏫 Derslik verilerini kullanma
- 📊 Rapor oluşturma

---

## 📁 Proje Yapısı

Projede kullanılan dosyalar:

- `test.cpp` → Ana program (C++ kodu)
- `ogrenciler.txt` → Öğrenci verileri
- `dersler.txt` → Ders bilgileri
- `derslikler.txt` → Derslik bilgileri
- `dersKaydi.txt` → Öğrenci ders kayıtları
- `rapor.txt` → Program çıktısı / rapor
